# AI Usage Report

## Tools Used & Use Cases

I used ChatGPT (Codex) to assist in planning and implementing the interactive features added in this assignment, including the name-based personalized greeting and the API-powered insight section. The tool provided guidance on structuring JavaScript logic for handling user input and dynamically updating content on the page.

It also helped me implement the API feature by suggesting the use of fetch with async/await, along with handling loading states and potential errors. This allowed me to create a more dynamic and user-responsive experience.

Additionally, AI supported me in refining the styling and improving small UI elements such as hover effects, as well as organizing and updating my documentation to maintain a clear and professional structure.

Overall, the tool was used as a supportive resource for enhancing interactivity and improving the overall user experience while keeping the core implementation under my control.

## Benefits & Challenges

Using AI helped speed up the development process by providing quick guidance on implementing JavaScript features and structuring interactive components. It made it easier to understand how to work with APIs and improve the responsiveness of the user interface.

One challenge I faced was ensuring that the API interactions were reliable and clearly communicated to the user, especially when handling loading states and errors. Additionally, I needed to ensure that the design remained simple and aligned with my original vision, as some suggestions were more complex than necessary.

## Learning Outcomes

Through this assignment, I learned how to handle user input events and dynamically update content based on user interaction.

I also gained experience using fetch with async/await to retrieve data from an external API, as well as implementing loading and error handling to improve user experience.

Additionally, I improved my ability to design interactive features while maintaining a clean and structured codebase.

## Responsible Use & Modifications

I carefully reviewed all AI-generated suggestions before applying them to my project. After implementing the suggested logic, I tested the features in the browser and modified them to ensure they worked correctly and aligned with my intended design.

I ensured that the final implementation reflected my own understanding by adjusting the structure, refining the behavior of interactive elements, and maintaining consistency with the assignment requirements.
